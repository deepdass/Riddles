# Riddles
A GUI Riddle Game made in python using TKinter

## Why
I use to enjoy solving Riddles online then i thought I could make this into a game and improve my coding

## How it Works 
- You get multiple Riddles to solve one after another and you need to answer them correctly to progress 
- there is also a skip button if you get stuck on a riddle
